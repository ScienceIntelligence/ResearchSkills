## Fact
As of late 2025 / early 2026, no published MCP (Model Context Protocol) server exists for land surface or atmospheric physics models (Noah-MP, WRF, CESM, NWM). Existing MCP servers for science cover only: quantum simulation (basic algorithms), Modelica simulations, computational chemistry tools, bioinformatics. The arXiv paper 2508.18489 (Aug 2025) surveys MCP for HPC/science but explicitly excludes atmospheric/land physics. A "first MCP for land surface models" would be a publishable contribution, with Noah-MP offering ~4,584 parameterization combinations that create a rich exploration landscape.
## Evidence
Multiple web searches (Oct 2025) for MCP + physics/earth-system models returned no hits; arXiv 2508.18489 scope explicitly excludes the domain; Noah-MP v5.0 (GMD 2023) refactoring enables modular access.
## Expiry Signal
When any MCP-for-LSM or MCP-for-atmospheric-model paper appears on arXiv/GMD/JAMES, this novelty claim expires. Monitor MCP blog posts and GMD submissions quarterly.
