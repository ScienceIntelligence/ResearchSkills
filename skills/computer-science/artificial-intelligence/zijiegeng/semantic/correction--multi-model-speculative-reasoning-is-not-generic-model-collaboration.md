## Fact
In the user's research framing, the key idea is not “multiple models collaborate” in a broad sense. The key idea is using multiple models for speculative reasoning: one or more models propose, verify, or accelerate reasoning trajectories under a speculative mechanism.

## Evidence
The user explicitly distinguished multi-model collaboration from the central mechanism: multi-model collaboration is the vehicle, while speculative reasoning is the scientific focus.

## LLM Default Belief
A frontier LLM may flatten this into “multi-agent collaboration” or “ensemble reasoning.”

That is the wrong framing here. The intended contribution is about speculative reasoning dynamics, not generic collaboration among agents.

## Expiry Signal
If the project later shifts toward negotiation, debate, role-specialized agents, or ensemble voting as the main contribution, this correction should be updated.
