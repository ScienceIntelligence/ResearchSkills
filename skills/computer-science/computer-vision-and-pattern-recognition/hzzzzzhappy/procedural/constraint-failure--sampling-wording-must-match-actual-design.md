## When + Exclusions
When revising methodology language from purposive sampling to random sampling.

Exclude cases where the sampling procedure is not described at all.

## Decision
Preferred: only state “random sampling” if participants were actually selected randomly from a defined population or sampling frame.

Rejected: mechanically replacing “purposive sampling” with “random sampling” for style or supervisor preference.

Reasoning: sampling terminology affects external validity claims. A false random-sampling claim is a methodological error, not a wording improvement.

## Local Verifiers
- Was there a sampling frame?
- Was a random selection method used?
- Were all eligible students equally likely to be selected?
- Does the final sample match the claimed population?

## Failure Handling
If the supervisor requires “random sampling” but the actual process was convenience-based, use cautious wording such as “a random sampling approach was intended” only if defensible, or revise the method transparently.

## Anti-exemplars
- Calling an online questionnaire sample random without random selection.
- Claiming representativeness from convenience recruitment.
- Changing sampling labels without changing the actual method.
